# cliurdu

An urdu dictionary utility on cli.
```
▶ cliurdu mukammal

===============================================================
		    Cliurdu v1.0.0
	A simple cli based utility for urdu dictionary
===============================================================


Hindi   :	मुकम्मल
Urdu    :	مُکَمَّل
Meaning :	complete,accomplished,perfect,
Source	:  	https://www.rekhta.org/
===============================================================
```

### Installation

```
python3 setup.py install
```
